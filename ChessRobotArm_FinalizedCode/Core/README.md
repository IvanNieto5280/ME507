@section overview Project Overview

The purpose of the chess robot arm was to be able to create an electromechanical system that could pick up chess pieces and be able to move them around a chess board by joystick manipulation of a user.

@section hardware Major Hardware

- Three Stepper motors

- Servo-based gripper

- Two Joysticks

- Emergency stop

@section software Software Implementation

-  Serial Perpherial Interface (SPI)

- Serial Wire Debug (SWD)

- Universal Synchronous Asynchronous Reciever-Transmitter (USART)
